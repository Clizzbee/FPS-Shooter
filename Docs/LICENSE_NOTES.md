Licensing notes
----------------
- Code is provided under the MIT license (see LICENSE).
- Art, sounds and third-party assets are NOT included in this scaffold. If you add assets:
  * Ensure you have explicit redistribution rights before putting the pack on the Marketplace.
  * Quixel Megascans included via the Epic Library are generally usable in Marketplace products, but verify terms.
  * For any purchased assets, keep receipts and license text and add a `ThirdPartyLicenses/` folder documenting sources.
- If you plan to resell on other storefronts, check each storefront's submission rules.
